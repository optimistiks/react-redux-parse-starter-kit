# react-redux-parse-starter-kit

Kickstart your application with React, Redux and Parse.

```npm run watch``` to start a webpack dev server.

```npm run build``` to build a production version.

work is in progress...

## What is included

* parse
* react
* redux
* react-router
* webpack
* babel (ES6 Stage 0)
* jest
* eslint
